# Transformers_Pro

This repository contains fine-tuning pipelines, and implementations related to Natural Language Processing using transformer architectures.

Main topics included:

- Transformer Architectures
- Text Summarization
- Sequence Classification
- Sequence Labeling
- Hugging Face Transformers
- NLP Evaluation Metrics
- Fine-Tuning Strategies
- Inference Pipelines

  Implemented Tasks

### Sequence Classification

Fine-tuning transformer models for tasks such as:

- Medicine prediction
- Text classification
- Biomedical NLP applications

### Summarization

Experiments using sequence-to-sequence architectures including:

- T5
- CNN/DailyMail dataset
- Automatic text summarization

### Sequence Labeling

Experiments related to:

- Named Entity Recognition (NER)
- Token classification
- Biomedical entity extraction
